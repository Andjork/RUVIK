# UNIAJC Educa Digital

Estructura inicial del proyecto para la plataforma de recursos educativos.

Estructura creada:
- `index.html`, `recursos.html`, `recurso-detalle.html`, `subir-recurso.html`
- `css/` con `styles.css`, `components.css`, `responsive.css`
- `js/` con `main.js`, `recursos.js`, `evaluacion.js`, `upload.js`
- `assets/` con `images/`, `docs/`, `videos/` (placeholders)
- `data/recursos.json` con ejemplo

Cómo probar localmente:
- Abrir `index.html` en el navegador (doble clic) o usar un servidor estático como `Live Server`.

Siguientes pasos sugeridos:
- Reemplazar los placeholders de imagen por archivos reales.
- Conectar un backend o API para almacenar recursos y valoraciones.
- Inicializar un repo Git y añadir .gitignore.
